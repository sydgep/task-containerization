# Task-level Containerization in Data Pipelines
This repository presents an experimental framework for evaluating task-level containerization as a design paradigm for data pipelines. 
It demonstrates how containerized tasks enable modular, reproducible, and portable workflows across heterogeneous environments. 
The repository includes both simple and healthcare-oriented pipelines implemented using Python, Prefect, and Docker, with comparative configurations ranging from monolithic scripts to fine-grained containerized tasks.
